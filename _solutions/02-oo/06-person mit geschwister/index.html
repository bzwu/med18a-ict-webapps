Beim Hinzufügen einer Person als Geschwister soll sichergestellt sein, dass bei der anderen Person die Liste von Geschwister auch aktualisiert wird.
Passen Sie danach den Code in index.js so an, dass zwei Personen (Male bzw. Female) erstellt werden. Fügen Sie die eine Person der anderen als Geschwister hinzu und geben Sie dann die Geschwister der anderen Person in der Ausgabe aus.
