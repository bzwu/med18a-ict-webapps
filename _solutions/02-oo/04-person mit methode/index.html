OOP Methoden

Erweitern Sie die Klasse Person von vorhin indem Sie eine Methode sayHello dazu programmieren.
Der Return-Wert der Methode soll ein Text sein, der die Eigenschaften in einem deutschen Satz enthält. Verwenden Sie für den Aufbau des Strings die neue Syntax `${value}`.
Passen Sie danach den Code in index.js so an, dass ein Person-Objekt erstellt wird, dann diese Methode aufgerufen und das Resultat auf die Konsole geschrieben wird.

Zusatzaufgaben
  • Anpassung der Methode «sayHello», sodass abhängig vom Geschlecht ein anderer Satz ausgegeben wird.
  • Anpassung des Konstruktors, sodass ein Geburtsdatum übergeben werden muss. Die Ausgabe soll sich nicht verändern – Berechnung des Alters im Konstruktor
