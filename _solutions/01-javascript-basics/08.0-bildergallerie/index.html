Bildergallerie.

Suchen Sie sich im Internet einige Bilder, die Ihnen gefallen. Erstellen Sie in Ihrem Code ein Array, das alle Adressen auf diese Bilder als String enthält.
Erstellen Sie für jedes der Bilder ein img-Tag als Unterelement von div#preview-image-container. Weisen Sie den so erstellten img-Tags die CSS-Klasse preview-image zu.
Programmieren Sie eine Function imagePreviewClickHandler. Bei Klick auf oben erstellte img-Tags soll diese Funktion aufgerufen werden.
In der Funktion soll das angeklickte Bild als grosses Bild angezeigt werden. Dazu muss das src-Attribute des Elements img#large-image mit dem Wert des src-Attributs des angeklickten Bildes (this.src) gesetzt werden.
Hinweise:

  • Element als Unter-Element hinzufügen
  • Bei einem Element eine CSS-Klasse hinzufügen / enfernen.
  • Events allgemein
  • Click-Event
