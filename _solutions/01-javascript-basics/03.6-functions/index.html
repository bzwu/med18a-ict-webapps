Funktionen als Variablen.

Programmieren Sie eine Funktion mit zwei Parametern: Eine Zahl und eine Funktion. Die Zahl gibt vor, wie oft die übergebene Funktion aufgerufen werden soll.
Die übergebene Funktion soll mit einer Zahl als Parameter aufgerufen werden. Programmieren Sie mehrere Versionen der Funktion mit dem Zahlenparameter, z.B. eine, die die übergebene Zahl quadriert und auf die Konsole ausgibt. Dann eine andere, bei der abhängig von der übergebenen Zahl ein unterschiedlicher Text auf der Konsole ausgegeben wird (z.B. wenn Zahl > 10, dann Text1, sonst Text2).
Nun rufen Sie die erste Funktion mit den zwei Parametern korrekt auf.
