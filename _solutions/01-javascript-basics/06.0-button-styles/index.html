Ausblenden und Umgestaltung von Buttons

Untenstehende Buttons sollen ihre Farbe verändern (rot werden), wenn mit der Maus drüber gefahren wird. Zudem sollen sie ausgeblendet werden, wenn man einen Button anklickt. Die Farbe eines Buttons soll von Rot auf Grün ändern, sobald die Maus den Button wieder verlässt (Stichwort mouseout). Zudem sollte bei Klick auf den "Alle ausblendeten Buttons wieder anzeigen" alle Buttons wieder anzeigen.
