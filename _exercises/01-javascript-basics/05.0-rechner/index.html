Einfacher Rechner

Erstellen Sie einen einfachen Rechner gemäss untenstehendem UI.

Mögliches Vorgehen:

  1. Unterscheiden Sie zuerst nicht, welcher Button gedrückt wurde - d.h. führen Sie diesselbe Logik aus - egal, welcher Button geklickt wurde.
  2. Lesen Sie bei einem solchen Klick einfach die beiden Eingabewerte und geben Sie die Zahlen mittels console.log(...) in der Konsole aus (F12-Browser-Tools)
  3. Wenn das läuft, erweitern Sie das Programm so, dass die beiden Eingabewerte addiert werden und danach im Resultats-Feld ausgegeben werden
  4. Wenn das auch läuft, bauen Sie mittels if oder switch-Statements die Logik ein, sodass bei Klick auf die Buttons die entsprechende Rechenoperation ausgeführt wird.
