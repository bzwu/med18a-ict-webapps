Lotto-Tipp Generator.

Programmieren Sie einen Generator für Lotto-Tipps. Der Tipp-Generator soll eine Funktion sein, die mit zwei Argumenten aufgerufen werden muss:

  • Anzahl zu erzeugender Zahlen (also z.B. 6)
  • Zahlenraum, aus dem die Zahlen erzeugt werden sollen (also, z.B. 45)
Die Funktion soll als Returnwert einen String zurückgeben, der alle generierten Zahlen komma-separiert enthält.
Nutzen Sie Math.random, um eine Zufallszahl zu erzeugen.
Programmieren Sie zudem den Code, um den Generator mit korrekten Werten aufzurufen und das Resultat auf die Konsole zu schreiben.
Hinweise:
  • Sie werden wahrscheinlich die Funktion parseInt benötigen.
  • Sie werden vielleicht auch die Array-Funktionen push, indexOf und join benötigen.

Wenn Sie das gemeistert haben:
Bestimmt ist Ihnen aufgefallen, dass teilweise Zahlen doppelt vorkommen - das geht bei einem Lotto-Tipp eigentlich nicht. Sorgen Sie dafür, dass Zahlen nicht doppelt aufgenommen werden.
