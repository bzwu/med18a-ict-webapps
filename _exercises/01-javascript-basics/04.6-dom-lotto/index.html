Grafischer Lotto-Tip Generator.

Die beiden Konfigurationswerte "Länge" und "Zahlenraum" sollen nun in Eingabefeldern vorgegeben werden. Beim Generieren des Lotto-Tips sollen diese eingegebenen Werte verwendet werden. Ein so eingelesener Wert ist immer ein Text und muss deshalb mittels parseInt zuerst in eine Zahl gewandelt werden.
Hinweise:

  • Element als Unter-Element hinzufügen
  • Bei einem Element eine CSS-Klasse hinzufügen / enfernen.
  • Events allgemein
  • Click-Event
