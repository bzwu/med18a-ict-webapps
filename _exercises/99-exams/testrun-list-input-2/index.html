Auflistung von Werten

Passen Sie das Programm von vorhin so an, dass...

  • neu eingegebene Werte immer zu oberst ausgegeben werden (Sortierung umkehren). Verwenden Sie dazu anstelle von parent.appendChild die Funktion parent.insertBefore.
  • der erste Wert immer fett dargestellt wird. Dafür gibt es drei M&öglichkeiten:
    1. Mittel Javascript: Setzen Sie entry.style.fontWeight = 'bold' bei jedem neu erstellten Eintrag. Beim zweitneusten Element muss der Inline-Style entfernt werden. Verwenden Sie dazu parent.firstChild.style.fontWeight = 'normal'
    2. Mittels Javascript + CSS: Erstellen Sie eine CSS-Klasse first-entry o.ä und fügen Sie diese CSS-Klasse jedem neu erstellten Element in Javavscript hinzu. Beim zweitneusten Element muss die CSS-Klasse entfernt werden. Verwenden Sie dazu parent.firstChild.classList.remove('first-entry')
    3. Mittels CSS: Schreiben Sie CSS, dessen Selektor nur das erste Element in der Liste adressiert.
