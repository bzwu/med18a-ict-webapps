Auflistung von Werten

Erstellen Sie ein Programm, über das in einem Eingabefeld Werte eingegeben werden können. Jeder eingegebene Wert soll in einer Liste ausgegeben werden.

  • Verwenden Sie var entry = document.createElement('div'), um die neuen Einträge zu erstellen
  • Verwenden Sie die Funktion parent.appendChild(entry) um das neue Element im DOM hinzuzufügen.
  • Weisen Sie den eingegebenen Text als Inhalt dem neu erstellten div hinzu (Verwenden Sie entry.innerText = input.value
  • Stellen Sie sicher, dass nur ein neuer Eintrag erstellt wird, wenn auch ein Text eingegeben wurde.
  • Nach der Übernahme des Texts in die Liste soll der Wert im Eingabefeld gelöscht werden (input.value = '').
  • Nutzen Sie inputField.focus(), um den Cursor nach jeder Eingabe direkt ins Eingabefeld zu setzen.
