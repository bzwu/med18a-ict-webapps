Übung Backofen

Ein Backofen lässt sich ein- und ausschalten. Er besitzt zudem eine max. Temperatur.
Wenn man ihn einschaltet, muss die Temperatur angegeben werden. Wenn diese Temperatur höher ist als die max. Temperatur, soll automatisch auf die max. Temperatur gestellt werden
Wenn man ihn ausschaltet, wird die Temperatur auf 0 zurückgesetzt.
Der Backofen soll eine Funktion haben, über diese man die aktuelle Temperatur auslesen kann.
Programmieren Sie eine Klasse «Backofen», in der die oben beschriebenen Anforderungen umgesetzt sind.
Danach erstellen Sie eine Instanz Backofen mit max. Temperatur 220°, schalten dann den Backofen mit einer Temperatur von 300° ein, lesen die aktuelle Temperatur des Backofens aus und schalten dann den Backofen wieder ab.
