OOP Vererbung

Erstellen Sie zwei Klassen Male und Female. Diese beiden Klassen sollen von Person erben.
Im Konstruktor der neuen Klassen soll auch der Konstruktor von Person aufgerufen werden.
In den neuen Klassen soll jeweils die Methode sayHello überschrieben werden, sodass die if-Anweisung in der sayHello-Methode der Klasse Person entfällt.
Passen Sie danach den Code in index.js so an, dass ein Female oder Male-Objekt erstellt wird und wieder derselbe Text in die Ausgabe geschrieben wird.
