Übung Erde

Unsere Erde besteht aus Kontinente und diese wiederum aus Länder.
Bilden Sie das in Klassen ab.
Wenn eine Instanz der Klasse "Land" erstellt wird, muss angegeben werden, wieviele Menschen dort wohnen (in Millionen).
Wenn eine Instanz der Klasse "Kontinent" erstellt wird, muss angegeben werden, welche Länder dazu gehören.
Wenn eine Instanz der Klasse "Erde" erstellt wird, muss angegeben werden, welche Kontinente dazu gehören.
Alle Klassen (Erde, Kontinent, Land) besitzen eine Methode "getAnzahlBewohner()"
Instanzieren Sie einige Länder, 2-3 Kontinente und ein Erde-Objekt. Danach rufen Sie die Methoden "getAnzahlBewohner" des Erde-Objekts auf.
