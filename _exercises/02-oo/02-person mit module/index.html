Erstellen einer ersten Klasse "Person"

Erstellen Sie eine Klasse Person mit den Eigenschaften name, age und gender.
Programmieren Sie die Klasse im File person.js

Wechseln Sie dann ins File index.js, importieren Sie Person und instanzieren Sie dort zwei Objekte der Klasse Person. Weisen Sie den Eigenschaften gültige Werte zu.

Danach soll der Altersunterschied der beiden Personen unten in das dafür vorgesehene Feld geschrieben werden (in der Art "Der Altersunterschied von Tim und Tom beträgt 3 Jahre.").
